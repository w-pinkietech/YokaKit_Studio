## 更新メモ
- 新しいドキュメントやディレクトリを追加したら、この index と `README.md` / `AGENTS.md` を同時に更新する。
- リンク切れがないかを `rg` やプレビューで確認し、必要に応じて Issue に記録する。
- 参照先を削除・移動した場合は、関連する README / AGENTS / INDEX から古いリンクを除去する。
